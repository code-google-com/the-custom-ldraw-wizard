This is the latest developent source code.

To install:

Download the NSIS (http://nsis.sf.net).

Compile CLPG.nsi.

Move the THE CONTENTS OF include-files to the Include directory in your NSIS installation path.

Put template\template.nsi in the same folder as CLPG.exe.

To use:

Run CLPG.exe.

Fill out the details.

Compile template.nsi
